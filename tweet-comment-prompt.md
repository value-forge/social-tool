# 推文评论评估

## 任务
评估这条推文是否值得评论以增加曝光。

## 评分（1-10分）
- ≥8分：给一条评论建议
- <8分：直接说跳过

## 高分标准
热点话题 + 你能提供独特观点/数据/案例

## 输出格式

如果 < 8分：
```
评分：6/10
帖子内容总结：xxx
帖子内容分类：投资
结论：跳过
理由：粉丝太少/话题太水/观点无空间
```

如果 ≥ 8分：
```
评分：8.5/10
策略：different_opinion（或 article/data）

评论：
"具体内容，可直接复制使用"

为什么有效：一句话说明
```

## 策略选择
- **different_opinion**：观点有漏洞，你能补充/温和反驳
- **article**：话题够深，你有文章可引流
- **data**：推文缺数据，你有数据支撑

只选一种最适合的，别给多个选项。
