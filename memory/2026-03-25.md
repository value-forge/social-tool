## Social Tool - Twitter Monitoring & Growth Platform

### Project Overview
Built a complete backend service for monitoring Twitter accounts and automated growth. The tool helps users track influencers, get real-time tweet notifications via DingTalk, and AI-powered content analysis.

### Architecture
- **Framework**: Go-Kratos (microservices)
- **Database**: MongoDB Atlas (M0 free tier)
- **Cache**: Redis (Upstash)
- **External APIs**: TwitterAPI.io
- **Notifications**: DingTalk webhook
- **AI**: Kimi API for tweet analysis

### Database Schema (4 Collections)

1. **users** - System users (operators)
   - Email, name, avatar
   - Settings: DingTalk webhook, thresholds
   - Notifications: DingTalk toggle, front-row alerts, quiet hours

2. **twitter_users** - Global Twitter accounts (shared)
   - twitter_id, username, profile, stats
   - ref_count (reference counting for lifecycle)
   - monitor_status, last_fetched_at

3. **user_monitored_accounts** - Watch relationships
   - user_id → twitter_user_id
   - Display config: group, tags, keywords
   - Status: active/paused

4. **tweets** - Tweet data + AI analysis
   - tweet_id, text, author snapshot
   - AI fields: category, sentiment, importance_score
   - is_front_row_opportunity (auto-calculated)
   - summary, topics, suggested_angles
   - TTL: 30 days auto-expire

### Key Features Implemented

1. **5-Minute Tweet Fetching**
   - Worker polls TwitterAPI.io every 5 minutes
   - Fetches latest tweets for all monitored accounts
   - Updates last_fetched_at timestamp

2. **AI Analysis Pipeline**
   - Auto-categorizes: 技术观点/产品更新/行业新闻/市场分析/闲聊
   - Sentiment analysis: positive/neutral/negative
   - Importance scoring: 0-10 (based on followers + content)
   - Front-row detection: >100k followers + <5min
   - Topic extraction + reply angle suggestions

3. **DingTalk Notifications**
   - 3 message templates:
     - 🚨 Front-row opportunity (countdown timer)
     - ⭐ High importance tweets (score ≥8)
     - Normal link cards
   - Keyword filtering per monitor
   - Quiet hours support
   - Global notification settings per user

4. **Smart Filtering**
   - Keywords: Only notify if tweet contains keywords
   - Importance threshold: Skip low-score tweets
   - Status check: Skip paused monitors
